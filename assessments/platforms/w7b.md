# W7b 平台作业（GWAS 关联测试 · Association Testing in GWAS）

| 平台 | 内容 | 目的 | 能力 |
|------|------|------|------|
| GWAS 基础概念概览 | 什么是 GWAS、SNP、等位基因频率、表型-基因型关联、单核苷酸多态性的统计学基础 | 建立对全基因组关联分析的整体认识；理解次要等位基因频率（MAF）在 SNP 选择中的角色；为 `b7-maf` 题包奠定基础 | GB2 |
| Frisch–Waugh–Lovell 残差化（FWL）与协变量调整 | 线性回归中的协变量、混杂变量、群体结构校正、主成分分析（PCA）为何用于校正；FWL 定理的数学直觉与实现细节 | 理解如何通过残差化处理协变量而不显式包含它们；为 `b7-assoc` 的实现提供理论支撑；掌握大规模 GWAS 中的快速关联测试策略 | GB2 |
| 多重假设检验与基因组控制（Genomic Control） | Bonferroni 矫正、FDR 控制、多重测试膨胀（inflation）、λ_GC 定义与计算、QQ 图绘制与解读、Manhattan 图的基本形式 | 理解为什么统计检验会由于群体结构/连锁不平衡/样本相关性而膨胀；学会用 λ_GC 检测系统偏差；掌握 QQ 图和 Manhattan 图作为质量控制工具的用法；为 `b7-gc` 题包提供背景支持 | GB2 |
| 逆正态变换（INT）在 GWAS 中的应用 | INT 定义、秩统计量、正态近似的好处、何时采用 INT；INT 在处理非正态分布表型（如计数表型、生存时间）中的优势 | 理解 INT 如何增强关联测试在偏斜表型上的稳健性；掌握 `rank_int()` 的数学实现；认识到正态化表型可以简化统计推理；为 `b7-int` 题包的设计背景提供支撑 | GB2 |

做法：先通过 GWAS 基础概念理解协会数据结构，再用 FWL 定理深化对协变量调整的理解，然后学习多重测试膨胀与基因组控制的原理，最后理解 INT 对非正态表型的稳健性改进。结合 `b7-maf`/`b7-assoc`/`b7-int`/`b7-gc` 题包的实现，完整掌握 GWAS 关联测试的核心工作流。
