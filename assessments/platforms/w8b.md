# W8b 平台作业（群体结构与线性混合模型 · Population Structure & Linear Mixed Models）

| 平台 | 内容 | 目的 | 能力 |
|------|------|------|------|
| 群体结构的影响与基因型主成分分析 | 什么是群体结构、与病杂交（admixture）、假关联（cryptic association）的产生机制；基因型 PCA 的原理与实现；使用 PC 作为协变量校正群体分层的方法；PC 数量的选择 | 理解为什么群体结构会导致关联信号的虚假膨胀；掌握基因型 PCA 如何捕捉群体间的遗传差异；学会用主成分作为关联模型的协变量进行群体结构校正；为 `b8-genopca` 题包提供理论基础 | GB3 |
| 线性混合模型与全基因组岭回归 | 线性混合模型的基本形式（固定效应 + 随机效应）；多基因随机效应的定义与性质；全基因组岭回归作为 LMM 多基因效应的闭式近似；岭参数（alpha）的选择；与传统单变量关联测试的区别 | 理解 LMM 如何通过随机效应模型化多基因背景；掌握岭回归的闭式解与数值稳定性；认识到岭回归是计算效率的多基因建模方法；为 `b8-ridge-lmm` 题包提供数学基础；为后续 REGENIE 两步法的学习铺垫 | GB3 |
| LOCO（留一染色体外型残差化）与 Proximal Contamination | LOCO 的原理与定义；proximal contamination 的产生机制（被检变异及其连锁变异在多基因背景中的泄漏）；LOCO 在避免虚假膨胀中的作用；LOCO 的计算流程与效率 | 理解为什么直接用全基因组岭回归预测的表型在被检变异所在染色体上会产生虚假信号；掌握 LOCO 通过排除目标染色体的 SNP 来消除此偏差的机制；学会实现 LOCO 残差化的标准工作流；为 `b8-loco` 题包的设计与验证提供支撑 | GB3 |
| REGENIE 两步法与大规模关联分析 | REGENIE 算法的两步框架（Step 1：全基因组岭回归 + LOCO 残差化；Step 2：对每个变异独立进行关联检验）；相比单变量关联测试的计算优势；参数设置与实践应用；REGENIE 与其他 LMM 工具的比较 | 理解如何通过分离多基因背景的估计与逐变异的关联检验来加速大规模 GWAS；掌握 REGENIE 两步法如何结合 LOCO 避免 proximal contamination；认识到REGENIE 在处理百万级 SNP 与大样本时的效率优势；为大规模基因组关联分析的实践应用做准备 | GB3 |
| PLINK/pgen 基因组格式与数据转换 | PLINK binary format (.bed/.bim/.fam) 的结构、每个文件的内容与作用；pgen/psam/pvar 格式（PLINK 2.0）的优势与使用场景；基因型数据的编码方式（0-2 计数等位基因）；格式转换的基本命令与工作流 | 了解常见的基因组关联分析数据格式；理解不同格式的压缩率与访问效率；学会用标准工具进行格式转换；为大规模 GWAS 数据处理与工具使用做准备 | GB4 |
| LMM 与 BOLT-LMM 论文 | BOLT-LMM 论文的核心贡献（分层岭回归 LOCO）；与传统混合模型（如 GREML）的关系与差异；大规模应用案例（UK Biobank 等）；LMM 在关联分析中的当代地位与挑战 | 通过阅读一篇关键文献，深化对 LMM 理论与实践的理解；认识到 LOCO 在避免偏差中的重要性；了解大规模 GWAS 中 LMM 的标准工作流与最佳实践 | GB4 |

做法：先从群体结构的影响与 PCA 校正建立直觉，再理解线性混合模型与岭回归的多基因建模思想，然后学习 LOCO 如何避免 proximal contamination 的虚假膨胀，最后结合 REGENIE 两步法的实际应用与 PLINK/pgen 格式的数据处理，形成完整的大规模 GWAS 关联分析工作流。结合 `b8-genopca`/`b8-ridge-lmm`/`b8-loco` 题包的实现与 w8b-lmm 测验，掌握群体结构校正、多基因建模与两步关联分析的核心原理与实践。
